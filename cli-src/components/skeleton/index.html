<tab-group> & <tab-item>

This example shows how to use children factory to create a pair of custom components working together.

Static (in html)

    <tab-group>
      <tab-item name="" active>...</tab-item>
      ...
    </tab-group>

This is a paragraph in "One" section

This is a paragraph in "Two" section

This is a paragraph in "Three" section

Click buttons to switch sections. Use developer tools to drag & drop or remove one of the <tab-item> in the <tab-group>.

Dynamic (in other custom component)

<tab-dynamic>...</tab-dynamic>

Click "Add item" button to create dynamic sections. Change active section and see changelog below the tab group (by the change custom event listener).

Use developer tools to drag & drop or remove one of the <tab-item> in the <tab-group>.
